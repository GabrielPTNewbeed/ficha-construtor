# ficha-construtor

D&D 5e character sheet for browser use

## Features
- Complete D&D 5e character sheet
- All 16 skills and 6 saving throws
- Spell management (9 levels + cantrips)
- Proficiencies, senses, resistances
- localStorage persistence
- JSON import/export
- Dark theme UI

## Getting Started

```bash
npm install
npm run dev
```

Visit `http://localhost:5173`
